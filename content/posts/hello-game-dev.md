## 왜 기록하는가

게임을 만들면서 부딪히는 문제들, 그리고 그걸 풀어낸 과정은 시간이 지나면 금방 잊혀집니다. 특히 언리얼 엔진으로 작업하다 보면 같은 실수를 몇 달 뒤에 다시 반복하는 경우가 많았습니다. 이 블로그는 그런 시행착오를 남겨두기 위한 공간입니다.

## 앞으로 다룰 주제

- **게임개발**: 언리얼 엔진, 네트워크 동기화, 게임플레이 프로그래밍
- **리버싱**: 바이너리 분석, 안티치트 우회/방어 연구
- **시스템**: 운영체제, 성능 최적화, 로우레벨 이야기

```cpp
// 첫 포스트 기념 코드 하이라이트 테스트
UCLASS()
class GOATBUTT_API AHeadbuttCharacter : public ACharacter
{
    GENERATED_BODY()

public:
    AHeadbuttCharacter();
};
```

코드 하이라이팅과 목차 기능이 잘 동작하는지 확인하기 위한 첫 글입니다. 앞으로 더 구체적인 개발 기록을 남길 예정입니다.
